Cascading Stylr Sheets-CSS

São folhas de estilo e cascata, com o objetivo de realizar a estilização de conteudos de documentos HTML.Podemos definir elas como:strong> Cores, fontes, tamanhos, sombras e posicionamnetos.

Existem 3 formas de utilizar o Css

Inline
Os codigos Css saõ definidos detro de uma tag Html e afetam somente a tag em que foi defeninda
Incorporado
Os codigos CSS são definidos no cabeçalho, dentro da tag head e afetam somente o documento que foi definido
Externo
Os cdigos Css são definidos em um arquivo externo e afetam todos os documentos que estão linkados ao CSS.
