Explicações das seguintes Tags:

Table
É usada para definições de tabelas, que são contruidas por linhas e colunas, que incluem todos os elementos, as tags de tabela é "table" abre e fecha e tambem representado as linhas por "tr" e a da coluna (celula) é a "td".
Del
Essa tag é usada para indentificar os textos excluidos em um documento porem ela mostra as modificações feita no documento, durante todo o projeto.
Ins
É utilizada junto coma teg DEL, ela é util para mostrar que um determinado texto foi modificado frequentemente.
Suo
Ele escreve textos com letras menores e com um aliamento a baixo dos outros conteudos.
Sup
Utilizado para escrever textos com letras menores e com um aliamento acima dos outros conteudos.
Blockquote
O Blockquote é indicado que o texto tem uma longa citação.
Q
Esse éindicado para uma citação de uma frase curta.
Abbr
é a tag que representa uma abreviaçao e opcionalmente fornece uma descrição completa.
Addrees
Fornece informações de contatos para os documentos.
Cite
Represnta uma referencia do trabalahndo, sendo de forma abreviada com o uso corretamente ou não.
Bdo
Usado para substituir a diricionalidade do texto atual.
Code
É utilizadado para a separação de textos normais e codigos, evitando a confusão dos textos escritos.
Pre
É a representação de um texto pré-formatado, que apresenta o texto conforme foi digitado no HTML.
Kbd
Utilizado para a definição da entrada dos dados do usuario, pelo teclado
